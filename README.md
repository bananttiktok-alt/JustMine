## IP Сервера

Ради безопасности сервера и наших игроков, категорически запрещается использовать сторонние лаунчеры. \
Единственный разрешенный лаунчер - JustMine Launcher (не модифицированный!) 

Доступ к серверу уже был настроен в Minecraft. Вам осталось только запустить игру.
## JustMine Launcher

Все ненавидят устанавливать новый Minecraft для каждого сервера самостоятельно. \
Встречайте, JustMine Launcher. \
\
Все моды уже на месте! Вам осталось только запустить игру. \
